**Информационная система «Учёт сотрудников»**
Данная система предназначена для автоматизации кадрового учёта в организации. Она позволяет управлять списком сотрудников, распределять их по отделам и разграничивать права доступа.

*Технологический стек*
Backend: Python (Flask), PostgreSQL, Psycopg2
Frontend: HTML5, CSS3, JavaScript (Vanilla JS)
API: RESTful API (JSON)

*Предварительные требования*
Перед запуском убедитесь, что у вас установлены:
Python 3.8 или выше.
СУБД PostgreSQL.
Библиотеки Python: flask, flask-cors, psycopg2-binary.

*Установка и настройка*

1. Настройка базы данных
Создайте базу данных в PostgreSQL и выполните SQL-скрипт для создания таблицы:
CREATE DATABASE company_db;
CREATE TABLE employees (
    id SERIAL PRIMARY KEY,
    last_name VARCHAR(100) NOT NULL,
    first_name VARCHAR(100) NOT NULL,
    department VARCHAR(100),
    email VARCHAR(150)
);

2. Конфигурация сервера
Откройте файл server/app.py и убедитесь, что параметры подключения в DB_CONFIG соответствуют вашим настройкам PostgreSQL (хост, имя пользователя, пароль).

3. Установка зависимостей
Выполните команду в терминале:
pip install flask flask-cors psycopg2-binary

*Запуск системы*
1. Перейдите в папку с проектом:
cd path/to/project/server

2. Запустите сервер:
python app.py

3. Откройте браузер и перейдите по адресу:
http://localhost:5000

*Роли пользователей*

В системе реализовано два уровня доступа:
-Администратор: Полный доступ (просмотр, добавление, редактирование, удаление).
-Пользователь: Только чтение списка сотрудников.
Переключение ролей осуществляется в верхней панели интерфейса.
